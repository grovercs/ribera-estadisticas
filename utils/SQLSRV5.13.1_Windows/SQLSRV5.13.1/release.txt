Release Information
Microsoft Drivers 5.13.1 for PHP for SQL Server
April 2026


INTRODUCTION
------------
This file contains late-breaking or other important information
that supplements the Microsoft Drivers for PHP for SQL Server
documentation. You should read this file completely before 
installing the PHP driver.

For the latest release notes, see:
https://learn.microsoft.com/sql/connect/php/release-notes-php-sql-driver

Your feedback is very important to us and we will strive
to respond to your feedback in a timely manner. For information
about providing feedback by using the PHP driver blogs and online
forums, see the Microsoft Drivers for PHP for SQL Server page at
https://learn.microsoft.com/sql/connect/php/microsoft-php-driver-for-sql-server

INSTALLATION
------------
Instructions for installing the PHP driver are located in SQLSRV_Readme.htm.
Refer to that file for information about installing the PHP driver.

For the latest installation instructions, see:
Windows: https://learn.microsoft.com/sql/connect/php/loading-the-php-sql-driver
Linux/macOS: https://learn.microsoft.com/sql/connect/php/installation-tutorial-linux-mac

SUPPORTED OPERATING SYSTEMS
---------------------------
For the latest system requirements, see:
https://learn.microsoft.com/sql/connect/php/system-requirements-for-the-php-sql-driver

For the latest support matrix, see:
https://learn.microsoft.com/sql/connect/php/microsoft-php-drivers-for-sql-server-support-matrix

The Microsoft Drivers for PHP for SQL Server supports the following operating systems:
	o Windows Server 2025
	o Windows Server 2022
	o Windows Server 2019
	o Windows Server 2016
	o Windows 11

The list above is an example of some of the supported operating systems.

RELEASE CONTENTS
----------------
The Microsoft Drivers for PHP for SQL Server zip file unpacks the following files in the specified locations, relative to the selected directory:

<selected directory>\Microsoft DRIVERS FOR PHP FOR MICROSOFT SQL SERVER - Standalone (free) Use Terms.txt
<selected directory>\release.txt
<selected directory>\SQLSRV_Readme.htm
<selected directory>\SQLSRV_ThirdPartyNotices.txt
For each supported version of PHP, the following files are included. Replace XY in the filename with the PHP major version (X) and minor version(Y).
<selected directory>\php_pdo_sqlsrv_XY_nts_x86.dll
<selected directory>\php_pdo_sqlsrv_XY_nts_x64.dll
<selected directory>\php_pdo_sqlsrv_XY_ts_x86.dll
<selected directory>\php_pdo_sqlsrv_XY_ts_x64.dll
<selected directory>\php_sqlsrv_XY_nts_x86.dll
<selected directory>\php_sqlsrv_XY_nts_x64.dll
<selected directory>\php_sqlsrv_XY_ts_x86.dll
<selected directory>\php_sqlsrv_XY_ts_x64.dll
